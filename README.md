# workshops

## Начало

Для того, чтобы приступить к первой тетрадке [intro.ipynb](intro.ipynb), прежде всего необходимо создать виртуальное окружение и установить необходимые библиотеки. Для этого выполняем команду:

```bash
python3 -m venv venv
```

Активируем наше виртуальное окружение

```bash
source venv/bin/activate
```

После этого необходимо обновить библиотеки, которые отвечают для установку других модулей

```bash
pip install -U pip wheel setuptools
```

Наконец, ставим все необходимое для работы. Далее мы будем уже использовать отдельный файл [requirements.txt](requirements.txt), но пока поставим все вручную

```bash
pip install jupyter scikit-learn pandas matplotlib
```

Если что-то понадобится еще, то всегда можно будет доустановить что-то в терминале (с активированным виртуальным окружением) или в самом Jupyter Notebook. После установки библиотек в последнем может потребоваться перезагрузить кернел тетрадки, для этого надо будет просто нажать Restart Kernel или на значок обновления. Важно! при этом все переменные внутри тетрадки сбросятся и надо будет выполнить все ячейки сначала. 

После установки библиотек запускаем jupyter. Он должен автоматически открыть в браузере необходимый интерфейс

```bash
jupyter-notebook
```
